    Facial Classifier

    This was truly my first end-to-end machine learning project. I began by building a computer vision model (CNN) which could predict the age, race, and gender of all individuals present in an image. I did so in PyTorch and trained my network on the VGGFace2 and UTKFace datasets, achieving an average 85% accuracy across the 3 dependents. I then used Flask and Bootstrap to build a web application around my model, allowing anyone to upload an image and see what my app thinks! Above is a screenshot of some predictions the app made.
